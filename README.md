# Anime Popularity Trend — Data Analysis & Machine Learning Project
This repository contains an ongoing research project analyzing anime popularity trends using the MyAnimeList (MAL) dataset. The project includes exploratory data analysis (EDA), data cleaning, preprocessing, baseline modeling, neural network development, scenario analysis, and dashboard visualizations.

# Project Purpose
The goal of this project is to investigate:
--What factors influence anime popularity
--How audience engagement relates to production characteristics
--Whether neural networks can predict popularity or score trends
--How different scenarios (genre changes, episode count, release year) affect outcomes

# Key Features Analyzed
Numerical:
Score

Categorical:
Type

Source

--These features were selected based on relevance to popularity, audience engagement, and production characteristics.
